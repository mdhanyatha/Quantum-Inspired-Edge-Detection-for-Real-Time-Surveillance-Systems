# Setup Instructions

1. Install Python 3.9 or higher.

2. Install required dependencies:
   pip install -r requirements.txt

3. Run the real-time edge detection system:
   python src/app.py

4. Generate performance graphs:
   python src/plot_graph.py

5. Press the following keys during execution:
   1 - Quantum Mode
   2 - Sobel Mode
   3 - Canny Mode
   p - Toggle PSNR
   r - Start/Stop Recording
   s - Save Frame
   q - Quit
